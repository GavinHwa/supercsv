Distribution

Super CSV is distributed as a .zip file containing various .jar files and full source code. Here is a short explanation of each .jar archive.
spiffy-with_source-all-x.xx.jarExternal dependency library that is highly re-useable in your own projects. See http://spiffyframework.sourceforge.net/
SuperCSV-x.xx.jarVersion x.xx of Super CSV. Only the compiled classes are in this archive.
SuperCSV-with_src-x.xx.jarVersion x.xx of Super CSV with the source code included. This integrates nicely with modern IDE's which allows one to quickly browse the code inside of the archive or extract the javadoc
SuperCSV-x.xx-source.zipThe source code only.
SuperCSV-1.50-javadoc.zipThe Javadoc documentation. Can be integrated with your favorite IDE
